Projects

  • Project1: Tele-dermatology application

    Skin conditions such as acne, rosacea, and pigmentation may not be life-threatening but can be very disturbing to the person experiencing it. For such benign skin conditions, people often avoid going to the dermatologist either due to monetary issues or due to lack of time. But if left untreated, these conditions could aggravate in time. The project aims at developing an android application that can be accessed anytime and anywhere for instant diagnosis and indicating the probability of skin abnormalities like rosacea, pigmentation, and acne. Once the abnormality has been detected, the application aims at recommending products for treating the same. It also shows nearby hospitals where the dermatology department is present on Google maps. Tensorflow Lite was used to deploy the app on the android device. The images were trained using a transfer learning approach and MobileNetV2 was used for the same. The system was able to detect the abnormalities viz: rosacea, pigmentation, and acne with an accuracy of 80% on the testing set. Tech stack used: Android Studio, Tensorflow Lite, CNN, Tensorflow, Keras, MobileNetV2 for transfer learning

  • Project2: Garbage Profiling System

    About 2 billion tons of garbage is generated annually and one-third of it is mishandled. A model developed using Mask R-CNN to aid the garbage profiling process was developed. The dataset used for this study was obtained from Kaggle and a few external sources. After cleaning and labeling the data, the model was trained using the Mask R-CNN algorithm to classify plastic waste. To make the model user-friendly, it was deployed on a website and the results were visualized using Python’s Pygal library.

  • Project4: Potato Disease Identification

    Potato harvest is affected mainly because of two common diseases- early blight and late blight, and because of this, farmers growing potatoes are facing a lot of economical loss. This project aims to detect these diseases in an early stage to prevent economical loss by accurately identifying the type of disease. Data cleaning and pre-processing was done using tf dataset and data augmentation. Following this, the model was built using CNN for image classification and then exported. For the mobile application part, this exported model was as converted into tflite model using quantization and then was deployed on google cloud to make the classification process more user-friendly.

  • Project8: Smart Medical Report generator

    During an accident or mishap, the victim may not be conscious which makes it difficult for doctors to know about his previous medical history. This system makes it easy to access the medical record of the victim. An RFID tag is assigned to each person. This tag will hold the previous medical history of the patient which can be updated regularly by the doctors. Only selected doctors whose records are stored will be able to view the patient's record. This system saves time and money which would have been invested by the patient if they start treatment of already existing ailment from scratch.

Papers published

  • Paper 1: A Skin Disease Detection System Using CNN Deep Learning Algorithm

    In this paper, a model developed using deep learning has been presented. This model distinguishes healthy skin from the skin having abnormalities such as acne, rosacea, and pigmentation and also suggests products for its treatment. A repertoire of 5000 images was made for this study through Dermnet.com and other external sources. Transfer learning has been used by extracting image features using a MobileNetV2 model which is pre-trained on over one million ImageNet images, then adding and training a fully connected layer with the dataset obtained to classify the skin into one of the four types. For a ubiquitous and user-friendly use of the presented model, it was deployed on an android app through TensorFlow Lite. The main functions of the mobile application are: (1) accurately detect the abnormality based on the surface of the skin scanned by the user. (2) Suggest treatment based on the abnormality detected. (3) List nearby hospitals for treatment.
